获取短消息内容接口

地址：

/MessageBox/List

方法：

GET

是否需要登录：

是

参数说明：

参数名称 是否必选 参数类型 参数描述
currentPageNumber 否 Integer 分页列表的当前页数default:1)
pageSize 否 Integer 当前页记录的条数(default:10)
type 否 Integer 消息的类型：{0:收件,1:发件,2:未读},默认(所有类型)
status 否 Integer 消息的状态：{0:已读,1:未读,2:接收者已删除,3:发送者已删除,4:已删除},默认(所有类型)

请求示例：

curl -d 'id=1' '/MessageBox/List'

返回值说明：

参数名称 是否必选 参数类型 参数描述
messageList 是 List<MessageBoxDto> 短消息列表
