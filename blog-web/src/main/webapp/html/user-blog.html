获取用户博客接口

地址：

/WebBlog/GetUserBlog

方法：

GET

是否需要登录：

否

参数说明：

参数名称 是否必选 参数类型 参数描述
userId 是 Integer 用户id
pageSize 否 String 获取数量，不填是10
offset 否 String 起始位子，因为是按id倒序，第一次不填，后面请求传入最后一条的数据id

请求示例：

curl -d 'userId=1&pageSize=5&offset=6' '/WebBlog/GetUserBlog'
