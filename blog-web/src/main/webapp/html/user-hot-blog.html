获取用户热门博客接口

地址：

/WebBlog/GetUserHotBlog

方法：

GET

是否需要登录：

否

参数说明：

参数名称 是否必选 参数类型 参数描述
userId 是 Integer 用户id
num 否 Integer 获取数量，不填是10

请求示例：

curl -d 'userId=1&num=5' '/WebBlog/GetUserHotBlog'
