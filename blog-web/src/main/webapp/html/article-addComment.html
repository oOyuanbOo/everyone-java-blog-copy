添加评论（回复）接口

地址：

/Article/AddComment

方法：

POST

是否需要登录：

是

参数说明：

参数名称 是否必选 参数类型 参数描述
comment 是 String 评论内容
articleId 是（添加评论时必须，添加回复时不必须） Integer 文章id
replyCommentId 是（添加回复时必须，添加评论时不必须） Integer 父级评论id

请求示例：

curl -d 添加评论：'articleid=1&comment=我是评论' 添加回复：'replyCommentId=1&comment=我是回复' '/Article/AddComment'

返回值说明：

参数名称 是否必选 参数类型 参数描述
commentId 是 Integer 新增加的评论id

返回示例：

{"code":200,"message":"请求成功","data":1}
